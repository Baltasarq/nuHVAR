nuHVAR

Resumen

Aplicación para Android orientada al análisis de la variabilidad del ritmo cardíaco.

Mediante el uso de una banda cardíaca o cualquier otro dispositivo Bluetooth, esta herramienta procesa el ritmo cardíaco durante un tiempo determinado por el usuario, produciendo gráficas y datos para favorecer el análisis de la variabilidad del ritmo cardíaco.
